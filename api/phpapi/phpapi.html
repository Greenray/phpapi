Class phpapi

public class phpapi

php tokenizer and parser.
Particularly the packages, classes and options specified by the user.
It is the root of the parsed tokens and is passed to the doclet to be formatted into output.
Program   phpapi: The PHP Documentation Creator
File   classes/phpapi.php
Version   4.0
Author   Victor Nabatov greenray.spb@gmail.com
Copyright   (c) 2015 Victor Nabatov
License   Creative Commons Attribution-NonCommercial-ShareAlike 3.0 Unported License
Overview   The main program package. It sets global variables, loads classes, tokenizes and parses php files. After that it creates descriptions of the program elements and calls doclet to create documentation pages.

public tag createTag (string $name, string $text, array &$data, rootDoc $root, mixed &$root)
Creates a tag.
This method first tries to load a Taglet for the given tag name, upon failing it
then tries to load a phpapi specialised tag class (e.g. classes/paramtag.php),
if it still has not found a tag class it uses the standard tag class.
Parameters string $name The name of the tag
  string $text The contents of the tag
  array &$data The reference to the doc comment data array
  rootDoc $root The reference to the root element
Return tag Tag object
