Class phpapi

  • phpapi

public class phpapi

php tokenizer and parser.
Particularly the packages, classes and options specified by the user.
It is the root of the parsed tokens and is passed to the doclet to be formatted into output.
Program   phpapi: PHP Documentation Creator
Version   6.0
Author   Victor Nabatov greenray.spb@gmail.com

File   /system/classes/phpapi.php
Overview   Main program package. It sets global variables, loads classes, tokenizes and parses php files. After that it creates descriptions of the program elements and calls doclet to create documentation pages.
