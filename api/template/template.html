Overview   Completely separated from php code. Handles special tags and replaces them with commands of the php interpreter. It is able recursively execute functions and directives: FOREACH, IF, ELSE, ELSEIF, SWITCH, CASE, BREAK, DEFAULT, INCLUDE, CONTINUE.
